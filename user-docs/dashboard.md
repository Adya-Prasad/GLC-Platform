# Dashboard User Guide

The Dashboard is your central command center for monitoring ESG performance and managing loan applications.

## 1. Overview

Upon logging in, you are greeted with the main dashboard. It provides a real-time snapshot of your environmental impact and application status.

## 2. Key Performance Indicators (KPIs)

At the top of the dashboard, you'll find critical metrics:

| Metric | Description |
| :--- | :--- |
| **ESG Score** | Your current composite ESG score (0-100). Higher is better. |
| **Active Loans** | Number of ongoing loan applications or active credit lines. |
| **Carbon Footprint** | Estimated CO2 emissions reduction (in tons). |
| **Pending Actions** | Tasks requiring your immediate attention (e.g., uploading docs). |

## 3. Navigation Menu

The sidebar provides access to all platform features:

- **Dashboard**: Return to this main overview.
- **New Application**: Start a new green loan request.
- **My Applications**: View detailed status of all your submissions.
- **Documents**: Manage your uploaded files and evidence.
- **Reports**: Generate and view ESG compliance reports.
- **Docs & Learn**: Access help guides and learning resources.

## 4. Recent Activity

The "Recent Activity" section shows a chronological log of important events:

- Application submissions
- Document upload confirmations
- Status updates from lenders
- New ESG score calculations

> **Tip**: Check this section daily to stay updated on your loan process.
