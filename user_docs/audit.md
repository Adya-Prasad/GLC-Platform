# Audit Report Guide

The Audit Report page provides comprehensive ESG analysis and compliance assessment for loan applications.

## 1. Overview

The Audit Report is the central hub for reviewing loan applications against Green Loan Principles (GLP) and Sustainability-Linked Loan Principles (SLLP).

## 2. Page Tabs

### Overview Tab

Displays key application information:
- **Project Details**: Name, sector, location, description
- **Loan Information**: Amount, currency, tenor, use of proceeds
- **Organization Snapshot**: Company name, contact info, annual revenue
- **Emissions Data**: Scope 1, 2, 3 emissions with visual breakdown

### ESG Analysis Tab

Comprehensive ESG scoring and compliance:

| Section | Description |
| :--- | :--- |
| **ESG Composite Score** | Overall score (0-100) with gauge visualization |
| **GLP Eligibility Status** | Whether project meets Green Loan Principles |
| **LMA Four Core Components** | Use of Proceeds, Project Evaluation, Management, Reporting |
| **SLLP Compliance** | KPI Selection, SPT Calibration, Loan Characteristics |
| **DNSH Assessment** | Do No Significant Harm criteria checklist |
| **Carbon Lock-in Risk** | Assessment of fossil fuel dependency risks |

### AI Insight Tab

AI-powered document analysis:

#### AI Agent
- Click **"Initiate AI Agent"** to analyze uploaded sustainability reports
- AI extracts key metrics, essential points, and answers LMA Framework questions

#### Data Performance Section
- **Essential Points**: Key findings with importance levels (Critical, High, Medium)
- **Quantitative Data**: Extracted metrics (emissions, revenue, targets)
- **Qualitative Information**: ESG practices and policies
- **LMA Framework Questions**: 
  1. Financial performance information
  2. Waste management practices
  3. Labor and employee practices
  4. Renewable energy usage or plans
  5. Environmental protection measures

#### Document Chat
- Ask questions about the uploaded documents
- AI provides answers with confidence scores
- Examples: "What are the emission targets?", "Describe renewable energy plans"

#### Save AI Report
- Click **"Save AI Report"** to generate a PDF
- PDF includes all AI analysis, metrics, and Q&A
- Saved to Loan Assets for future reference

### Decision Tab (Lender Only)

Lender decision-making tools:

| Action | Description |
| :--- | :--- |
| **Update Status** | Change application status (Pending, Under Review, Approved, Rejected) |
| **Add Notes** | Record reviewer observations and feedback |
| **Schedule Review** | Set follow-up review dates |
| **Request Documents** | Ask borrower for additional documentation |
| **Generate Report** | Create formal assessment report |

> **Note**: Borrowers can view the Decision tab but cannot take actions.

## 3. ESG Score Calculation

The ESG Composite Score is calculated from multiple factors:

| Factor | Weight | Description |
| :--- | :--- | :--- |
| **Questionnaire** | 30% | ESG compliance questionnaire responses |
| **GLP Alignment** | 25% | Use of proceeds alignment with GLP categories |
| **DNSH Compliance** | 15% | Do No Significant Harm assessment |
| **Sector Risk** | 10% | Industry-specific ESG risk level |
| **Data Completeness** | 15% | How complete the application data is |
| **Emissions Reporting** | 5% | Quality of emissions data provided |

## 4. GLP Eligibility

A project is GLP Eligible if:
- Use of proceeds aligns with recognized green categories
- Project demonstrates clear environmental benefits
- DNSH criteria are satisfied
- No significant carbon lock-in risks

## 6. Tips for Using Audit Report

- **Lenders**: Run AI Agent first to get comprehensive document analysis
- **Review Essential Points**: Focus on Critical and High importance items
- **Check GLP Compliance**: Ensure all four core components are addressed
- **Use Document Chat**: Ask specific questions for deeper insights
- **Save Reports**: Generate PDF reports for record-keeping and sharing
