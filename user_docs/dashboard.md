# Dashboard User Guide

The Dashboard is your central command center for monitoring loan applications and ESG performance on the GLC Platform.

## 1. Overview

Upon logging in, you are greeted with the main dashboard tailored to your role (Borrower or Lender).

## 2. Lender Dashboard

Lenders see a comprehensive portfolio overview with key metrics and all loan applications.

### Key Metrics Cards

| Metric | Description |
| :--- | :--- |
| **Total Applications** | Total number of loan applications in the system |
| **Total Loan Pool** | Sum of all requested loan amounts |
| **Approved** | Number of approved applications |
| **Pending Review** | Applications awaiting lender decision |
| **Avg ESG Score** | Average ESG score across all applications |
| **Green Eligible** | Applications meeting GLP eligibility criteria |

### Status Breakdown

Visual progress bars showing distribution of applications by status:
- Pending (Yellow)
- Under Review (Blue)
- Approved (Green)
- Rejected (Red)

### Sector Distribution

List of sectors with application counts, helping identify portfolio concentration.

### Financed Emissions

Total estimated CO2 emissions (tCO2e) from approved loans for carbon accounting.

### Loans Table

Complete table of all applications with columns:
- Organization Name
- Project Name
- Sector
- Loan Amount
- Annual Revenue
- ESG Score
- Status

> **Tip**: Click any row to navigate directly to the Audit Report for detailed analysis.

## 3. Borrower Dashboard

Borrowers see their own applications and submission options.

### Quick Actions
- **New Application**: Start a new green loan request
- **My Applications**: View all your submitted applications

### Application Cards

Each application shows:
- Project name and loan ID
- Requested amount
- Current status with color indicator
- ESG score (if calculated)

## 4. Navigation Menu

The sidebar provides access to all platform features:

| Menu Item | Description |
| :--- | :--- |
| **Dashboard** | Return to main overview |
| **New Application** | Start a new green loan request (Borrower) |
| **All Applications** | View all applications (Lender) / My Applications (Borrower) |
| **Loan Assets** | Manage uploaded documents and AI reports |
| **Audit Report** | Detailed ESG analysis and compliance review |
| **Docs & Learn** | Access help guides and documentation |

## 5. Tips for Effective Use

- **Lenders**: Use the ESG Score column to quickly identify high-performing applications
- **Borrowers**: Check the status regularly and respond to any document requests promptly
- **Both**: The Audit Report page provides the most detailed analysis for any application
